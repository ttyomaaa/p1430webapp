Процедура расшифрования:
Пример сообщения:
1230 = 3tle = 1tl = 250 = WZA UHL =
JKGLP SQTGS ABTRQ PTWYB LENDP

Сообщение отправлено в 12:30 = Состоит из 3 частей(3 teile) = Это первая часть сообщения(1 teil) = Она состоит из 180 символов = Стартовая позиция + ключ сообщения = Идентификационная группа

Расшифрование(процедура с 1940 г.) происходило так:
 1. Диски машины устанавливаются в состояние, указанное в заголовке сообщения – WZA;
 2. Вводится зашифрованный ключ сообщения – UHL;
 3. В результате получается ключ сообщения – SXT;
 4. Диски устанавливаются в состояние SXT;
 5. Первые 5 букв сообщения отбрасываются – это буквенная идентификационная группа;
 6. Вводится полученное сообщение;
 7. В результате получается открытый текст.
