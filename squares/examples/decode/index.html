Пример расшифрования:
Рассмотрим следующий пример заполнения квадратов.

Расшифруем сообщение “:УТБУЦХХЪРТЬ”. Разбиваем сообщение на биграммы: :У ТБ УЦ ХХ ЪР ТЬ. В правом квадрате находим первую букву биграммы, в левом – вторую.
По диагоналям прямоугольника находим соответствующую паре :У пару ПА.

Паре букв ТБ соответствует пара букв РО. Аналогично расшифровываем остальные биграммы.
Получим открытое сообщение: “ПАРОЛЬ: КИНО”.
