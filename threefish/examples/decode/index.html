Алгоритм расшифрования:
Для алгоритма Threefish процедура расшифрования обратна процедуре шифрования. Раундовые ключи используются в обратном порядке, и каждый раунд состоит из обратных операций. Вместо функции \(MIX_{d,j}\) используется функция \(deMIX_{d,j}\), которая выполняет операции вычитания по модулю \(2^{64}\) и побитовый циклический сдвиг вправо. Этапы каждого раунда расшифрования также выполняются в обратном порядке.
